# mini_agent — Architecture & State

A coding agent powered by DeepSeek V4 Pro. Terminal REPL + Textual TUI + Electron app.
717 tests, 31 tools, SQLite memory.

## Current Architecture

```
mini_agent.py        Terminal REPL entry point
tui.py               Textual TUI (AgentWorker, themes, tree, diff preview)
electron_app/        Electron desktop app (main.js, preload.cjs, index.html)
electron_bridge.py   JSON-RPC bridge for Electron (stdin/stdout)
llm.py               LLM turn orchestration (_execute_tools, _inject_context, run_agent_turn)
api.py               API calls (call_deepseek, truncate_content, format_tool_detail)
prompt.py            System prompt + .mini_agent.rules injection
config.py            AgentConfig (.env + TOML + env + CLI priority)

## Key State (2025-07-17)

### API Keys & Security
- NO hardcoded keys in source. All keys via .env file (DEEPSEEK_API_KEY, EXA_API_KEY, OPENAI_API_KEY).
- .env is .gitignore'd. Priority: CLI > env var > .env > TOML > default.
- max_tokens lowered 800k → 200k for cost savings.

### Inter-Agent Communication (FULLY IMPLEMENTED)
- 9 message types: text, handoff.result, handoff.request, handoff.ack, status.heartbeat, status.error, coord.fan_out, coord.fan_in, coord.sync
- Tools: agent_message, agent_read, agent_handoff, agent_inbox, agent_subscribe
- Routing: direct target delivery, subscription-based, broadcast
- Sub-agents instructed to send heartbeats every ~3 turns + check inbox each turn
- Shared file coordination rules in sub-agent system prompt

### File Reservation System (FULLY IMPLEMENTED)
- _FILE_RESERVATIONS dict (path → task_id) with threading.Lock
- _current_agent_id (threading.local) set per sub-agent
- write_file and edit_file BOTH check reservations and block cross-agent collisions
- reserve_file(), release_file(), release_all_files() in tools/__init__.py
- AgentRuntime releases all files on task completion

### Electron App
- Working JSON-RPC bridge (init, chat, cancel methods)
- Token streaming from backend to frontend
- CommonJS preload.cjs (ESM preload.js did not load)
- Uses venv/bin/python3 (system python3 lacked deps)
- CSP allows unsafe-inline scripts

### Configuration
- .env file for API keys (recommended, gitignored)
- .mini_agent.toml for advanced settings (MCP servers, overwrite flags)
- Priority: CLI flags > env vars > .env file > .mini_agent.toml > defaults
