# mini_agent — Electron Desktop App

A terminal-style AI coding assistant with a React frontend, Electron shell, and Python backend.

## Quick Start

### Prerequisites

| Tool | Version | Install |
|------|---------|---------|
| **Python** | 3.10+ | [python.org](https://www.python.org/downloads/) |
| **Node.js** | 18+ | [nodejs.org](https://nodejs.org/) |
| **ripgrep** | any | `brew install ripgrep` (macOS) / `apt install ripgrep` (Linux) |

### One-step install

```bash
# From the mini_agent project root:
bash setup.sh
```

This runs all checks, creates the Python venv, installs npm packages, and builds the renderer.

### Launch

```bash
cd mini_agent_electron
npm start
```

## API Key Setup

On first launch, if no API key is detected, the app will show a **Settings panel** where you can enter your key. Supported providers:

| Provider | Environment Variable |
|----------|---------------------|
| DeepSeek | `DEEPSEEK_API_KEY` |
| Claude   | `CLAUDE_API_KEY`    |
| xAI      | `XAI_API_KEY`       |
| Ollama   | `OLLAMA_API_KEY`    |

Keys are saved to `~/.mini_agent_env` and persist across restarts.

Alternatively, set one before launching:
```bash
export DEEPSEEK_API_KEY=sk-your-key-here
```

## Development Mode

Run the renderer with hot-reload and DevTools:

```bash
cd mini_agent_electron
npm run dev          # launches Electron + Vite dev server
```

Or run just the Vite dev server (no Electron):
```bash
npm run dev:renderer
```

## Architecture

```
mini_agent_electron/
├── main.js              # Electron main process — spawns Python backend, IPC bridge
├── preload.js           # Context bridge (contextIsolation-safe renderer ↔ main IPC)
├── package.json         # Node.js deps + scripts
├── vite.config.js       # Vite config for React renderer build
├── backend/
│   └── server.py        # Python JSON-lines server (stdin/stdout protocol)
└── renderer/
    ├── index.html        # Entry HTML
    ├── dist/             # Built React app (generated by `npm run build`)
    └── src/
        ├── main.jsx      # React entry point
        ├── App.jsx       # Root component
        ├── components/   # UI components
        └── hooks/        # Custom React hooks
```

**How it works:**
1. Electron launches → spawns `python3 backend/server.py` as a child process
2. Renderer sends user messages via IPC → main process → Python stdin (JSON-lines)
3. Python streams tokens/events back via stdout → main process → renderer via IPC events

## Troubleshooting

### "Backend server.py not found"
Run from the `mini_agent_electron/` directory. The main process expects `backend/server.py` relative to `main.js`.

### Blank window on launch
The renderer wasn't built. Run `npm run build` first, or use `npm start` (which auto-builds if `dist/` is missing).

### Python import errors
Make sure you ran `bash setup.sh` from the project root. The Python backend needs packages from `../requirements.txt` installed in the venv.

### API key not detected
- Keys in `~/.mini_agent_env` take priority over shell environment variables
- Use the in-app Settings panel (⚙️ icon) to save keys — they persist automatically
